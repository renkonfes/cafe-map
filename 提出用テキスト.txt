アピールポイント
Leaflet.jsを使って、カフェの場所を地図上にピンで表示できるようにしました。
地図をクリックすると緯度・経度が自動で入力され、カフェ名、住所、営業時間、Wifi、電源、駐車場、コメントを登録できます。
また、Supabaseと接続することで、登録したカフェ情報が自分のブラウザだけでなく他の人のブラウザからも見られるようにしました。
メールアドレスとパスワードでの会員登録・ログイン機能を追加し、ログインした人だけがカフェ登録、コメント、いいねをできるようにしています。
スマートフォンでも見やすいように、画面幅に合わせて地図と入力フォームが縦に並ぶデザインにしました。

使用した技術
HTML / CSS / JavaScript / Leaflet.js / Supabase / GitHub Pages

発表で話す内容メモ
このアプリは、行ってよかったカフェを地図上に記録して共有するために作りました。
最初はlocalStorageでも保存できますが、それだと自分のブラウザにしかデータが残らないため、最終版ではSupabaseを使ってデータを共有できるようにしました。
苦労した点は、地図のピン、入力フォーム、Supabaseのデータを連動させる部分です。
今後改善するなら、写真投稿機能や、駅名・特徴で検索できる機能を追加したいです。
