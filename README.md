# カフェマップ

## アプリの概要

カフェの場所を地図上に登録して、他の人とも共有できるWebアプリです。
地図をクリックすると緯度・経度が入力され、カフェ名、住所、営業時間、Wifi、電源、駐車場、コメントを登録できます。

## 作成した理由

授業で作成した地図アプリをもとに、実際に使いやすいカフェ共有アプリに発展させたいと考えました。
カフェを探すときは、場所だけでなく、Wifiや電源、営業時間なども知りたいと思ったため、それらの情報も登録できるようにしました。

## 主な機能

1. Leaflet.jsを使った地図表示
2. 地図クリックによる緯度・経度の自動入力
3. カフェ情報の新規登録・表示・編集・削除
4. Supabaseを使ったデータ共有
5. メールアドレスとパスワードによるユーザー登録・ログイン
6. コメント投稿
7. いいね機能
8. スマートフォンでも見やすいレスポンシブデザイン

## 工夫した点

最初に授業で扱ったlocalStorageでは、自分のブラウザの中にしかデータを保存できません。
そこで、最終版ではSupabaseを使い、他の人のブラウザからも同じカフェ情報を見られるようにしました。

また、ログインしている人だけがカフェ登録やコメント投稿をできるようにし、誰が登録したデータなのかを管理できるようにしました。
地図をクリックすると緯度と経度が自動で入るようにしたため、手入力のミスも減らせるようにしています。
